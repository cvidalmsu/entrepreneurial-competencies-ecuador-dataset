# Entrepreneurial Competencies and Operational Risk in Ecuadorian Micro-Enterprises: A Dataset (2023--2024)

**Authors:** Jazmín Pérez-Salazar, Nicolás Márquez, Cristian Vidal-Silva  
**Year:** 2025  
**Journal:** Data (MDPI)  

## Description
This repository contains the dataset and metadata supporting the research article "Entrepreneurial Competencies and Operational Risk in Ecuadorian Micro-Enterprises". The data characterizes the competency profiles of 687 micro-entrepreneurs in Zone 5, Ecuador.

## Files Description
* `micro_enterprises_data.csv`: The primary raw dataset containing 687 observations and 11 variables.
* `metadata.csv`: Data dictionary describing the variable names, types, and coding schemes.
* `data_generation.py`: Python script used to ensure data consistency and statistical alignment.
